Hello! Geyword is a Google like keyword search through all registered stores based on graph database (Neo4j) to maintain schema graph. We use API to get the query string from the client, and sent this query to our Neo4j database. After the Keyword search finishing working on the server, the result containing column, database, record and table is returned.
